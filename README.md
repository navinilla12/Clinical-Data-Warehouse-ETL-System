#  Clinical Data Warehouse & ETL Reporting System

##  Overview
This project simulates a real-world healthcare data engineering pipeline inspired by clinical research and enterprise BI systems. It demonstrates ETL processing, data warehousing concepts, and KPI reporting using Python.

The system processes patient-level data, performs transformation and cleaning, builds a star-schema style data warehouse, and generates business intelligence insights.

---

##  Features
- End-to-end ETL pipeline (Extract, Transform, Load)
- Clinical-style data cleaning and preprocessing
- Star schema data warehouse simulation
- KPI generation for healthcare analytics
- SQL-style aggregations using Pandas
- BI-style visualizations
- Power BI-ready output datasets

---

##  Tech Stack
- Python (Pandas, NumPy)
- Matplotlib
- Data Warehousing Concepts
- ETL Pipeline Design
- Star Schema Modeling

---

##  Dataset
Public dataset used from:
- diabetes dataset (healthcare simulation data)

---

##  Pipeline Architecture

1. Data Extraction (CSV load)
2. Data Cleaning (missing value handling, transformation)
3. Data Modeling (Fact & Dimension tables)
4. KPI Calculation (business metrics)
5. Aggregation & Analysis
6. Export for BI tools (Power BI ready CSVs)

---

##  Key KPIs
- Total Patients Processed
- Disease Outcome Rate

- Age-based Health Risk Trends
